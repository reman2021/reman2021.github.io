Windows软件使用分享

发表于 2022-07-19 更新于 2022-08-15 分类于 软件 Waline： 本文字数： 968 阅读时长 ≈ 1 分钟

分享个人使用的 Windows 软件。

福昕高级PDF编辑器

PDF 是 Portable Document Format 的简称，意为“可携带文档格式”，是由 Adobe 公司开发的一种通用的文档格式，具有通用性好、保真性强等优点。
但是由于 PDF 本身的特点，选择一款稳定、快速的软件用以阅读编辑显得尤为重要。
目前较为推荐的有国内福昕和 Adobe 公司的软件，分别是福昕高级PDF编辑器和 Adobe Acrobat Reader DC。如果只有阅读需求，Microsoft Edge 的 PDF 阅读体验也是可以的。 在尝试过两个之后，个人还是更喜欢福昕高级PDF编辑器。


这里附上 9.7.1 版本的安装文件和来自胡萝卜周的补丁：
链接：https://pan.baidu.com/s/12zbUdP_Bsz65t1omdXKdLA
提取码：w899

知云文献翻译

外文论文文献翻译利器。
官网下载：https://www.zhiyunwenxian.cn/

EV录屏

官网下载：https://www.ieway.cn/evcapture.html

Mathpix Snipping Tool

识别公式并生成相应代码，非常好用。（每个月只能免费使用10次）
官网下载：https://mathpix.com/

Snipaste

个人最常用的截图工具，小巧好用。
官网下载：https://www.snipaste.com/

它还可以将截图贴在屏幕上，并支持放大缩小。


Bandizip

压缩解压缩软件，没有 WinRAR 烦人的广告。
官网下载：https://www.bandisoft.com/bandizip/

Wox + Everything

高效办公好助手。（用惯的话）
具体参考：Wox+Everything结合使用

向日葵远程控制

可用手机平板远程控制电脑，进行文件传输，免费版功能基本满足所需。
官网下载：https://sunlogin.oray.com/

PicPick

小工具箱。
官网下载：https://picpick.app/zh/

Linux Reader

适用于双系统，可在 Windows 下访问 Linux 的文件。
官网下载：https://www.diskinternals.com/download/

Motrix

简洁美观的磁力下载器，同样适用于 Linux、macOS。
官网下载：https://motrix.app/
